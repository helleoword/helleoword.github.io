Project

Image Compression Processing Experiment

We employed two techniques for image compression:

	1. SVD (Singular Value Decomposition): We applied SVD to compress images by decomposing the matrix formed from the image's RGB values into singular values, U, and V matrices. After reconstruction, we adjusted the ratio of singular values to observe its effect on image quality. The results demonstrated that by retaining 20% of the singular values, we could preserve most of the essential image information, yielding a satisfactory visual outcome.
	
	2. PCA (Principal Component Analysis): We used PCA for image compression by standardizing the image matrix and selecting principal components that capture the majority of the variance. By ignoring components that contribute less, we achieved compression. Our results indicated that retaining $50$ principal components produced a satisfactory image quality.
	
	The entire implementation was carried out in Python.

House price prediction based on linear regression algorithm

We formulated a regularized loss function to evaluate the model’s performance in predicting house prices, and used batch gradient descent for iterative optimization. The predicted price curve was then compared with the actual house prices to assess the model’s accuracy. Overall, the predictions were highly accurate. The final implementation was done in Python.

Handwritten digit recognition for digits 0 and 1 based on logistic regression

We utilized the Sigmoid function to estimate the probabilities for handwritten digit classification and employed Binary Cross-Entropy Loss (BCELoss) to evaluate the model’s performance. Gradient descent was used as the optimization method for iteratively updating the model’s parameters. After training, the model was tested on a set of images, achieving a test accuracy of 100%. The entire implementation was done in Python.

Handwritten digit recognition based on Softmax

The objective of this task is to apply the Softmax function for the classification of handwritten digit data. We utilized gradient descent as the optimization algorithm to iteratively update the model parameters, and cross-entropy loss as the metric to assess the performance of the model. After training, we visualized the results of the iterative process and evaluated the model’s performance on test images. The model was trained for 2000 iterations, achieving a test accuracy of 93.4%. The implementation was carried out in Python.
